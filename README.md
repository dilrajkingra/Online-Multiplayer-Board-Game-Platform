# Online-Multiplayer-Board-Game-Platform
Modular online board game platform supporting Chess, Go, and Tic-Tac-Toe with matchmaking, lobbies, ranking, and a GUI. Implements full game rules (checkmate, special moves, captures, Ko), 1v1 queueing, lobby control, ELO-style leaderboard, and prototype auth. Built with clean subsystem architecture for easy extension and testing tools.
